# Portfolio

白と黒を基調とした建築設計図風のデザインで、Webアプリケーションとモバイルアプリケーションの作品を展示するポートフォリオサイトです。

## 技術スタック

- **Next.js 16** (App Router)
- **TypeScript**
- **TailwindCSS 4**
- **SSG** (Static Site Generation) で高速化

## 機能

- トップページ：ヒーローセクションと作品プレビュー
- 作品一覧ページ：全作品をグリッド表示
- 作品詳細ページ：個別作品の詳細情報と画像ギャラリー
- Aboutページ：自己紹介とスキル
